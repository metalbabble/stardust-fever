# Assets

This folder contains game assets for Stardust Fever including:
- Images used for in-game graphics
- bgm (background music)
- Extras (not directly used by app, just contains more game assets if needed)

# Background Music

These are named in a predictable pattern: m1.mp3, m2.mp3, and so forth. The game will select a random track to play - just set the MUSIC_COUNT value in config.js to the highest-numbered file.
