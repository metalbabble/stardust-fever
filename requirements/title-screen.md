# Title Screen

The purpose of this screen is to show the user the title of the game with some quick instructions before they start playing...

# What appears on the title screen:

- In large text, the title "STARDUST FEVER"
- Below that "Use left/right arrow keys to rotate ship. Shoot asteroids and aliens - but try and collect "atoms" to recieve various power ups! How long can you survive!?
- And below that "Press ENTER to start a new game"

# Other notes

As the instructions indicate, pressing enter starts a new game - see [main-gameplay.md](main-gameplay.md)
